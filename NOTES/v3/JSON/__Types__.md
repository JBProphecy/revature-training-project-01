## **JSON Data Types**

There are exactly six data types in `JSON`:

- `object`
- `array`
- `string`
- `number`
- `boolean`
- `null`

---

## **JSON Rules**

In `JSON`, numbers cannot have leading `0`'s.

---

## **Python Equivalent**

| `JSON` Type | `Python` Type   | `Mapping`               |
| ----------- | --------------- | ----------------------- |
| `object`    | `dict`          | direct                  |
| `array`     | `list`          | direct                  |
| `string`    | `str`           | direct                  |
| `number`    | `int` / `float` | `Python` will determine |
| `boolean`   | `bool`          | direct                  |
| `null`      | `None`          | direct                  |
