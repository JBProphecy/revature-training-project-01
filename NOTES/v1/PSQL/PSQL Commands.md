# For Me

Use `psql --username postgres` to initiate login.

---

# Index

| Command                                | Description                                       |
| -------------------------------------- | ------------------------------------------------- |
| `\?`                                   | displays a list of commands                       |
| `\c`                                   | displays information about the current connection |
| `\c [database]`, `\connect [database]` | connects to a database                            |
| `\d`                                   | displays a list of tables                         |
| `\d+`                                  | for additional information                        |
| `\d [name]`                            | displays information about a table                |
| `\d+ [name]`                           | for additional information                        |
| `\l`, `\list`                          | displays a list of databases                      |
| `\q`                                   | exits the PSQL terminal                           |

---

# To Create a New Database

Use `CREATE database [database_name];` in the PSQL terminal
