## Primitive Data Types

There are seven primitive data types in JavaScript.
The names of these data types are lowercase.

1. string
2. number
3. boolean
4. undefined
5. null
6. symbol
7. bigint

---
## Non-Primitive Data Types

There are several reference data types in JavaScript.
The first letter in the names of these data types are capitalized.

- Object
- Array
- Function
- Date
- RegExp
- Map
- Set
- WeakMap
- WeakSet
