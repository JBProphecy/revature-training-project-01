## Console

The console is an object.
The console prints/outputs/displays information from JavaScript.
You can send information to the console by using this function...

---

$$ \text{console.log()} $$

---

**Example Code**
1. console.log("Hello World");
2. let myName = "Jack";
3. console.log(myName);

**Example Console**
1. Hello World
2. Jack
