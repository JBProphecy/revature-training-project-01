# All Annotations

| Annotation        | Purpose                                                                     |
| ----------------- | --------------------------------------------------------------------------- |
| `@Autowired`      | to achieve dependency injection                                             |
| `@Bean`           | to declare a bean in a `@configuration` class                               |
| `@Configuration`  | to declare a class that defines beans                                       |
| `@Component`      | to declare a class as a Spring component                                    |
| `@ComponentScan`  | to scan for stereotype annotations for the whole tree, starting from a base |
| `@Controller`     | a specialization of `@Component` for controller classes                     |
| `@GetMapping`     |                                                                             |
| `@Repository`     | a specialization of `@Component` for repository classes                     |
| `@RestController` |                                                                             |
| `@RequestMapping` |                                                                             |
| `@Service`        | a specialization of `@Component` for service classes                        |
| `@Scope`          | [[@Scope]]                                                       |
| `@Qualifier`      | to specify which bean should be `@autowired`                                |

# Stereotype Annotations

| Annotation    | Purpose                                                 |
| ------------- | ------------------------------------------------------- |
| `@Component`  | to declare a class as a Spring component                |
| `@Controller` | a specialization of `@Component` for controller classes |
| `@Repository` | a specialization of `@Component` for repository classes |
| `@Service`    | a specialization of `@Component` for service classes    |
