## Hot Keys

- up arrow $\;\Longrightarrow\;$ command history ~ back (older)
- down arrow $\;\Longrightarrow\;$ command ~ next (more recent)

- backspace $\;\Longrightarrow\;$ delete a character on the left side of the cursor
- delete $\;\Longrightarrow\;$ delete a character on the right side of the cursor

- left arrow $\;\Longrightarrow\;$ move cursor to the left by one character
- right arrow $\;\Longrightarrow\;$ move the cursor to the right by one character

- ctrl + left arrow $\;\Longrightarrow\;$ move the cursor to the left by one word
- ctrl + right arrow $\;\Longrightarrow\;$ move the cursor to the right by one word

- home $\;\Longrightarrow\;$ move the cursor to the beginning of the line
- end $\;\Longrightarrow\;$ move the cursor to the end of the line

- ctrl + home $\;\Longrightarrow\;$ delete everything you've typed on the left side of the cursor
- ctrl + end $\;\Longrightarrow\;$ delete everything you've typed on the right side of the cursor

## Updated Key Words

- CLS
	- Clears the screen.

- DIR = "Directory"
	- Displays a list of files and subdirectories in a directory.

- CHDIR = "Change Directory"
- CD = "Change Directory"
	- Displays the name of or changes the current directory.

- MKDIR = "Make Directory"
- MD = "Make Directory"
	- Creates a directory.

- RMDIR = "Remove Directory"
- RD = "Remove Directory"
	- Removes (deletes) a directory.

- TREE
	- Graphically displays the folder structure of a drive or path.

- COLOR
	- Sets the default console foreground and background colors.

## Change Directory (cd)

- You can move into a directory by typing "cd *name*" ~ replacing *name* with the name of the directory.

- You can move up or out of a directory by typing "cd .."

---

- You can move into multiple directories at once by typing "cd *name1*\\*name2*" ~ replacing *name1* and *name2* with the names of the directories, respectfully.
	- This line will take you into two directories, but you can do more if you please.

- You can move out of multiple directories at once like by typing "cd ../.."
	- This line will take you out of two directories, but you can do more if you please.

---

- If you press the tab key, you can cycle through the objects in the current working directory. If you type anything before you press tab, only the objects starting with what you've typed will be cycled, which means if you don't type anything before you press tab, you can cycle through all the objects in the current working directory.
