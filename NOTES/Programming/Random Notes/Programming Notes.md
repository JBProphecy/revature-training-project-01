# Strong vs Weak Typing

- For a strongly-typed language, the rules for working with types are strict.
- For a weakly-typed language, the rules for working with types are flexible.

---
# Static vs Dynamic Typing

- For a statically-typed language, types are checked at compile time.
- For a dynamically-typed language, types are checked at runtime.

---
# Authentication vs Authorization

**`Authentication`** involves the verification of credentials for a certain thing.

**`Authorization`** involves verifying that an authenticated user has certain permissions.
