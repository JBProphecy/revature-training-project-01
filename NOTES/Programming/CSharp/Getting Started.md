# Start a New Project

You can use a terminal to create a folder and then navigate inside of it. You'll need to enter the directory where you want to store your project and then run the following commands.

``` shell
mkdir a_name_for_your_project # to create a folder in the current directory
cd a_name_for_your_project # to enter the new folder/directory
```

Run the following command.

``` shell
dotnet new console
```
