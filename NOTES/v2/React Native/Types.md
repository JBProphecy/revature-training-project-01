# **React Native Element Prop Type**

``` ts
import { FlatListProps, ImageProps, ScrollViewProps, TextInputProps, TextProps, ViewProps } from "react-native"
```

| Element       | Prop Type          |
| ------------- | ------------------ |
| `FlatList<T>` | `FlatListProps<T>` |
| `Image`       | `ImageProps`       |
| `ScrollView`  | `ScrollViewProps`  |
| `Text`        | `TextInputProps`   |
| `View`        | `ViewProps`        |

---

# **React Native Element Style Prop**

| Element | Style Prop   |
| ------- | ------------ |
| `Image` | `ImageStyle` |
| `Text`  | `TextStyle`  |
| `View`  | `ViewStyle`  |
