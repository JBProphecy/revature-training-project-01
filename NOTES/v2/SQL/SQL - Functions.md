# Table of Aggregate Functions in SQL

| Aggregate Function | Description                         |
| ------------------ | ----------------------------------- |
| `COUNT()`          | returns the number of rows          |
| `SUM()`            | returns the sum of some values      |
| `AVERAGE()`        | returns the average of some values  |
| `MIN()`            | returns the largest of some values  |
| `MAX()`            | returns the smallest of some values |
| `FIRST()`          | returns the first of some rows      |
| `LAST()`           | returns the last of some rows       |

---

# Table of Scalar Functions in SQL

| Scalar Function | Description                                          |
| --------------- | ---------------------------------------------------- |
| `UCASE()`       | returns the uppercase version of a string            |
| `LCASE()`       | returns the lowercase version of a string            |
| `MIN()`         | returns the substring of a given string              |
| `LEN()`         | returns the length of a given string                 |
| `NOW()`         | returns the current date and time of the system      |
| `ROUND()`       | used to round a number to a number of decimal places |
| `FORMAT()`      | used to format a string                              |
