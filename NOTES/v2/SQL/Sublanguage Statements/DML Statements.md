# INSERT

**Insert Row**

``` sql
INSERT INTO table_name (column_names)
VALUES (column_values);
```

---
# UPDATE

**Update Data**

``` sql
UPDATE table_name
SET column_name = value
WHERE condition;

UPDATE table_name
SET column_name_01 = value_01, column_name_02 = value_02
WHERE condition;
```

---
# DELETE

**Delete Data**

``` sql
DELETE FROM table_name
WHERE condition;
```
