# Table of Layers

| Name              | Description                                                                                                          |
| ----------------- | -------------------------------------------------------------------------------------------------------------------- |
| Web Layer         | handles requests and responses - uses controllers to process requests                                                |
| Service Layer     | encapsulates the business logic of an application, serving as a link between the web layer and the data access layer |
| Data Access Layer | interacts with databases and external data sources                                                                   |
