# Naming Convention

For naming type parameters, the convention is to use single, uppercase letters.

| Generic      | For                         |
| ------------ | --------------------------- |
| E            | Element                     |
| K            | Map Key                     |
| V            | Map Value                   |
| N            | Number                      |
| T            | Generic Data Type           |
| S, U, V, ... | Multiple Generic Data Types |
