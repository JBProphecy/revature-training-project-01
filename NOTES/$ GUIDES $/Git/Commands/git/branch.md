
---

```
git branch
```

This command will display a list of local branches.

---

```
git branch -r
```

This command will display a list of remote branches.

---

```
git branch -a
```

This command will display a list of all branches, both local and remote.

---

```
git branch -d <branch_name>
```

This command will delete an existing branch safely.

---

```
git branch -D <branch_name>
```

This command will delete an existing branch forcefully.
