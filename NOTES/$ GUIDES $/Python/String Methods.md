## `str.is...()`

| Method               | Purpose                                                       |
| -------------------- | ------------------------------------------------------------- |
| `str.isalnum()`      | to check if all the characters in a string are alpha-numeric  |
| `str.isalpha()`      | to check if all the characters in a string are alphabetic     |
| `str.isascii()`      | to check if all the characters in a string are ASCII          |
| `str.isdecimal()`    | to check if all the characters in a string are decimal digits |
| `str.isdigit()`      | to check if all the characters in a string are digits         |
| `str.isidentifier()` | to check if a string is an identifier                         |
| `str.islower()`      | to check if all the characters in a string are lowercase      |
| `str.isnumeric()`    | to check if all the characters in a string are numeric        |
| `str.isprintable()`  | to check if all the characters in a string are printable      |
| `str.isspace()`      | to check if all the characters in a string are whitespace     |
| `str.istitle()`      | to check if a string is title-case                            |
| `str.isupper()`      | to check if all the characters in a string are uppercase      |
