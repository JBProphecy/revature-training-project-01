
---

```
pip --version
```

---

```
pip install package_name
```

This command will install a package.

---

```
pip install package_name==package_version
```

This command will install a specific version of a package.

---

```
pip install -r requirements.txt
```

This command will install all the packages listed in a file called `requirements.txt`.

---

```
pip list
```

This command will display a list of the packages installed in the current environment in a human-readable format.

---

```
pip freeze
```

This command will display a list of the packages installed in the current environment in a system-readable format.

---

```
pip freeze > requirements.txt
```

This command will save a list of the packages installed in the current environment in a file called `requirements.txt`
