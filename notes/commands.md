# **Commands**

These are some useful commands.

---

## **Normal Environment**

| Command | Purpose |
| - | - |
| `./venv/Scripts/Activate` | to activate the virtual environment |

---

## **Virtual Environment**

| Command | Purpose |
| - | - |
| `deactivate` | to deactivate the virtual environment |
| `pip list` | to display a list of the installed packages in a human-readable format |
| `pip freeze` | to display a list of the installed packages in a system-readable format |
| `pip freeze > requirements.txt` | to save a list of the installed packages in `requirements.txt` |
| `pip install -r requirements.txt` | to install the packages listed in `requirements.txt` |
| `pip install <package_name>` | to install a package |
| `pip uninstall <package_name>` | to uninstall a package |
| `pytest` | to run tests |
| `pytest -s` | to run tests and have console logs enabled |
